¿Qué es?

DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, based upon the foundations of progressive enhancement, and will add advanced interaction controls to any HTML table.

¿Cómo usarlo?

  1. instalar git
  2. git pull del repositorio: 
    > git clone git@github.com:miguelfg/jpd16-workshop-datatables.git
  3. instalar servidor local
    • python 2.X: 
      > python -m SimpleHTTPServer 8000
    • python 3.X: 
      > python -m http.server 8000
    • npm
  4. ejecutar servidor local desde la raíz del repositorio
  5. visitar url: http://localhost:xxx/src/example-1/
  6. [OPT] instalar csvkit
